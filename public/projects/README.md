# Previews de proyectos

Coloca aquí una captura por proyecto con **exactamente** estos nombres. En cuanto
el archivo exista, la card lo muestra automáticamente; si falta, la card cae con
gracia a su diseño de gradiente (no se rompe nada).

| Archivo            | Proyecto              | De dónde sacarla                          |
| ------------------ | --------------------- | ----------------------------------------- |
| `movieview.png`    | MovieView             | https://movieview-one.vercel.app          |
| `conecta.png`      | Conecta (privado)     | Captura difuminada / mockup sin datos reales |
| `ntt-data.png`     | Prueba Técnica NTT    | https://prueba-tecnica-nttdata.vercel.app |
| `kiichain.png`     | KiiChain              | Demo local o screenshot del repo          |
| `backend-jwt.png`  | Backend CRUD + JWT    | Captura de Postman / Swagger / terminal   |
| `appnotitas.png`   | Appnotitas            | Demo local de la app de notas             |

## Recomendaciones

- **Formato:** PNG o JPG. Idealmente `.webp` para que pesen menos.
- **Tamaño:** ~1200×800 px (ratio 3:2). Se recortan con `object-cover` desde arriba.
- **Peso:** < 300 KB cada una (optimízalas en https://squoosh.app).
- **Conecta es privado:** usa un mockup o difumina datos sensibles antes de subir.

> Las cards aplican un velo oscuro encima, así que no te preocupes si la captura
> es muy clara: el título y el texto siguen legibles.
